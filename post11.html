NVIDIA Docker 配置教程

详细介绍 NVIDIA Container Toolkit（NVIDIA Docker）的安装和配置方法，让 Docker 容器能够访问 GPU 资源。

网络知识库 网络技术学习平台

NVIDIA Docker 简介

NVIDIA Container Toolkit（原 NVIDIA Docker）允许您在 Docker 容器中使用 NVIDIA GPU 资源。它提供了一套工具和库，使得在容器中运行 GPU 加速的应用程序变得更加简单。

Docker 安装

卸载旧版本

如果之前安装过旧版本 Docker，需要先卸载： 

sudo apt-get remove docker docker-engine docker.io containerd runc
彻底卸载 Docker 引擎： 
sudo apt-get purge docker-ce docker-ce-cli containerd.io docker-compose-plugin
sudo rm -rf /var/lib/docker
sudo rm -rf /var/lib/containerd

自动安装脚本（推荐）

使用 Docker 官方提供的自动安装脚本： 

export DOWNLOAD_URL="https://mirrors.tuna.tsinghua.edu.cn/docker-ce"
curl -fsSL https://get.docker.com/ | sh

APT 安装

sudo apt-get install apt-transport-https ca-certificates curl gnupg2 software-properties-common
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://mirrors.tuna.tsinghua.edu.cn/docker-ce/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
sudo apt-get update
sudo apt-get install docker-ce

验证 Docker 安装

sudo docker run hello-world
如果输出 "Hello from Docker!"，说明 Docker 安装成功。

NVIDIA 显卡驱动安装

检查系统支持的驱动

ubuntu-drivers devices

安装 NVIDIA 驱动

sudo add-apt-repository ppa:graphics-drivers/ppa
sudo apt-get install dkms build-essential
sudo apt install nvidia-driver-525
sudo reboot
nvidia-smi
验证驱动是否成功加载： 
nvidia-smi

NVIDIA Container Toolkit 安装

添加 NVIDIA 软件源

distribution=$(. /etc/os-release;echo $ID$VERSION_ID)
curl -fsSL https://nvidia.github.io/libnvidia-container/gpgkey | sudo gpg --dearmor -o /usr/share/keyrings/nvidia-container-toolkit-keyring.gpg
curl -s -L https://nvidia.github.io/libnvidia-container/$distribution/libnvidia-container.list | \
  sed 's#deb https://#deb [signed-by=/usr/share/keyrings/nvidia-container-toolkit-keyring.gpg] https://#g' | \
  sudo tee /etc/apt/sources.list.d/nvidia-container-toolkit.list

安装 NVIDIA Docker 2 包

sudo apt-get update
sudo apt-get install -y nvidia-docker2
安装完成后重启 Docker 服务： 
sudo systemctl restart docker

验证 NVIDIA Docker 安装

运行基本的 CUDA 容器来验证： 

sudo docker run --rm --gpus all nvidia/cuda:12.2.91-base-ubuntu22.04 nvidia-smi
如果能够正常输出 NVIDIA-SMI 信息，说明 NVIDIA Docker 配置成功。

在容器中使用 GPU

运行支持 CUDA 的容器时，需要使用 --gpus 参数： 

sudo docker run --rm --gpus all nvidia/cuda:12.2.91-base-ubuntu22.04 nvidia-smi
sudo docker run --rm --gpus all nvidia/cuda:12.2.91-cudnn8-runtime-ubuntu22.04 python -c "import torch; print(torch.cuda.is_available())"

常用配置选项

指定 GPU 设备

指定使用特定 GPU 设备： 

sudo docker run --rm --gpus "device=0" nvidia/cuda nvidia-smi
或者使用 PCI 总线 ID： 
sudo docker run --rm --gpus "device=0000:01:00.0" nvidia/cuda nvidia-smi

指定 GPU 数量

sudo docker run --rm --gpus 2 nvidia/cuda nvidia-smi

常见问题

docker: Error response from daemon: could not select device driver "nvidia" with capabilities: [gpu].

说明 NVIDIA Docker 没有正确安装。解决方法：

  • 重新安装 NVIDIA Docker
  • 确保 Docker 服务已重启
  • 检查 NVIDIA 驱动是否正常工作

GPU 设备访问权限问题

如果容器无法访问 GPU 设备，检查权限设置： 

ls -l /dev/nvidia*
groups $(whoami)
确保用户在 docker 组中。

升级和维护

升级 NVIDIA Docker

sudo apt-get update
sudo apt-get upgrade nvidia-docker2

查看日志

journalctl -u docker.service --no-pager
