Mellanox CX-5 网卡配置流程

详细介绍 Mellanox CX-5 网卡的驱动安装、配置和测试方法，包括 IB 和 ROCE 模式。

网络知识库 网络技术学习平台

Mellanox CX-5 网卡简介

Mellanox CX-5 是一款高性能的网络适配器，支持多种网络技术，包括 InfiniBand (IB) 和 RoCE (RDMA over Converged Ethernet)。它提供高达 100Gbps 的数据传输速率，适用于高性能计算和数据中心应用。

驱动安装与环境配置

检查网卡识别状态

使用 lspci 命令检查网卡是否被系统识别： 

lspci | grep -i mellanox
输出类似于： 
01:00.0 Network controller: Mellanox Technologies MT27800 Family [ConnectX-5]

下载与安装驱动

从 Mellanox 官方网站下载驱动：Mellanox OFED 驱动 下载适合您系统的版本（建议选择稳定版本）。 

sudo chmod -R 777 MLNX_OFED_LINUX-5.8-4.1.5.0-ubuntu20.04-x86_64
cd MLNX_OFED_LINUX-5.8-4.1.5.0-ubuntu20.04-x86_64
sudo ./mlnxofedinstall
如果遇到依赖包问题，执行： 
sudo apt remove 包名称
# 或
sudo dpkg -r mft-repo-amd64-ubuntu2004-local mft-pcap mft-meta

检查网卡状态

ibstat
如果输出 State: UP 表明网卡端口可用，Link layer: Ethernet 表明处于 ROCE 模式。

手动激活网卡端口

如果网卡状态为 State: down, physical state: Linkup，说明端口处于禁用状态，手动激活： 

ibdev2netdev
sudo ip link set 网卡号 up

检查 PCIe 带宽

确保网卡 PCIe 带宽预留足够： 

dmesg | grep mlx5_core
确认是否有 PCIe BW: xx GB/s 信息，对于 100Gbps 网卡，至少需要 16x Gen3 或 8x Gen4 带宽。

配置网卡 IP 地址

sudo ifconfig 网卡号 IP地址 netmask 子网掩码
例如： 
sudo ifconfig enp1s0f1np1 10.0.0.1 netmask 255.255.255.0
重启网络管理器： 
sudo /etc/init.d/network-manager restart

网络性能测试

设置服务端

ib_send_bw -a -c RC -d mlx5_1 -i 1
选项说明：
  • -a：自动探测最大包大小
  • -c RC：使用可靠连接模式
  • -d：指定网卡设备
  • -i：指定端口号

客户端配置 MTU

sudo ifconfig enp1s0f1np1 mtu 4096

性能测试

# 服务端
ib_send_bw -a -c RC -d mlx5_1 -i 1
# 客户端
ib_send_bw -a -c RC -d mlx5_1 -i 1 10.0.0.7
测试命令可根据需要调整参数： 
ib_send_bw -c RC -s 163840 -d 网卡号 -i 1 -x 3 -D 1 -q 50 -t 1000 --run_infinitely

RoCE 模式配置

启用 RoCE

使用 mlnx_qos 工具启用 RoCE： 

sudo mlnx_qos -i 1 --enable_roce

配置 GID 索引

ibv_devinfo
查看 PORT_GIDS 信息。

常见问题

驱动安装失败

  • 确保下载的驱动与内核版本匹配
  • 更新系统内核到最新版本
  • 检查依赖包是否安装完整

网卡不被识别

  • 检查 PCIe 插槽连接是否正常
  • 在 BIOS 中启用 PCIe 设备
  • 尝试使用不同的 PCIe 插槽

性能不达标

  • 确认网卡运行在正确的 PCIe 模式
  • 检查 MTU 配置是否正确
  • 确认网络设备支持相应的速率

管理工具

  • ibstat：检查网卡状态
  • ibv_devinfo：获取详细设备信息
  • mlnx_qos：QoS 和 RoCE 配置
  • ib_send_bw：带宽测试工具
  • ib_send_lat：延迟测试工具

最佳实践

  • 使用 Mellanox 官方推荐的驱动版本
  • 保持系统内核和驱动版本匹配
  • 使用 MTU 9000（Jumbo Frame）提高性能
  • 对于高性能计算应用，建议使用 RDMA
