Markdown 语法使用说明

详细介绍 Markdown 语法的各种使用方法，包括标题、段落、文字格式、列表、表格、代码块等。

网络知识库 网络技术学习平台

一、标题

Markdown 支持 6 级标题，使用 # 符号表示，# 数量对应标题级别。

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

快捷键

  • Ctrl+数字 1~6：快速设置对应级别的标题
  • Ctrl+0：设置为普通文本
  • Ctrl+加号/减号：调整标题级别

二、段落

1. 换行

在 Markdown 中，段落之间需要空一行。如果需要强制换行，可以在行尾添加两个空格或使用 <br> 标签。

2. 分割线

使用三个或更多的星号、减号或下划线来创建分割线：

---
***
___

三、文字显示

1. 字体样式

  • 粗体：使用 **粗体** 或 __粗体__
  • 斜体：使用 *斜体* 或 _斜体_
  • 粗斜体：使用 ***粗斜体*** 或 ___粗斜体___
  • 删除线：使用 ~~删除线~~
  • 下划线：使用 <u>下划线</u>
  • 高亮：使用 ==高亮==

2. 上下标

x^2^  
H~2~O

效果：x2 和 H2O

四、列表

1. 无序列表

使用星号、加号或减号作为列表标记：

* 第一项
* 第二项
* 第三项

+ 第一项
+ 第二项
+ 第三项

- 第一项
- 第二项
- 第三项

2. 有序列表

使用数字加英文句点：

1. 第一项
2. 第二项
3. 第三项

3. 嵌套列表

使用缩进表示嵌套关系：

1. 第一项
   * 子项 1
   * 子项 2
2. 第二项
   1. 子项 1
   2. 子项 2

4. 任务列表

使用 - [ ] 表示未完成的任务，- [x] 表示已完成的任务：

- [ ] 未完成任务
- [x] 已完成任务

五、区块显示

使用 > 符号表示区块引用：

> 这是一个区块引用
>
> > 嵌套区块引用

六、代码显示

1. 行内代码

使用反引号包裹代码：

`int a = 0;`

2. 代码块

使用三个反引号包裹代码块，并可以指定语言：

```javascript
function hello() {
    console.log("Hello, World!");
}
```

七、链接

Markdown 支持两种链接方式：

[链接文字](链接地址 "可选标题")

<链接地址>

八、脚注

使用脚注可以为文本添加注释或参考资料：

这是一个技术[^1]

[^1]: 这是一个非常好用的框架。

九、图片插入

图片语法与链接类似，只是在前面多了一个感叹号：

![图片描述](图片路径 "可选标题")

十、表格

Markdown 支持简单的表格语法：

| 表头 1 | 表头 2 | 表头 3 |
| :--- | :---: | ---: |
| 左对齐 | 居中 | 右对齐 |
| 内容 1 | 内容 2 | 内容 3 |

十一、流程图

使用 Mermaid 语法可以绘制流程图：

1. 横向流程图

```mermaid
graph LR
A[方形]==>B(圆角)
B==>C{条件a}
C-->|a=1|D[结果1]
C-->|a=2|E[结果2]
F[横向流程图]
```

2. 竖向流程图

```mermaid
graph TD
A[方形]==>B(圆角)
B==>C{条件a}
C-->|a=1|D[结果1]
C-->|a=2|E[结果2]
F[竖向流程图]
```

十二、表情符号

Markdown 支持表情符号：

:happy:、:cry:、:man:

十三、数学公式

使用 LaTeX 语法可以插入数学公式：

1. 行中公式

$公式$

2. 独立公式

$$
公式
$$

常用公式示例

  • 分数：$\frac{a}{b}$ → $\frac{a}{b}$
  • 开方：$\sqrt{2}$ → $\sqrt{2}$
  • 积分：$\int_{a}^{b} f(x) dx$ → $\int_{a}^{b} f(x) dx$
  • 求和：$\sum_{i=1}^{n} i$ → $\sum_{i=1}^{n} i$
  • 极限：$\lim_{x \to \infty} f(x)$ → $\lim_{x \to \infty} f(x)$

十四、支持的 HTML 元素

Markdown 支持直接使用 HTML 标签：

  • 文本居中：<center>内容</center>
  • 快捷键：<kbd>Ctrl+C</kbd>
  • 加粗：<b>加粗</b>
  • 倾斜：<i>倾斜</i>
